# 📌 Attendance Management System

## 📖 Overview
This is a web-based application developed to automate and simplify student attendance management in educational institutions.

## 🚀 Features
- Admin, Teacher, and Student login system
- Online attendance marking
- Attendance percentage calculation
- Excel & PDF report generation
- Role-based dashboards

## 🛠️ Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Java (JSP, Servlets, JDBC)
- Database: MySQL
- Server: Apache Tomcat

## 🧩 Modules
### 👨‍💼 Admin
- Manage students and teachers
- View reports

### 👨‍🏫 Teacher
- Mark attendance
- Generate reports

### 🎓 Student
- View attendance and percentage

## 📊 Key Highlights
- Reduces manual work
- Improves accuracy
- Real-time data access

## 🔮 Future Scope
- Mobile app integration
- Face recognition attendance
- Cloud deployment
